Wax

A minimal computing publishing tool

What is Wax?

  • Leveraged Jekyll and Github Pages
  • Uses IIIF (International Image Interoperability Framework)(why IIIF?)
  • Uses OpenSeaDragon viewer but can use any other
  • Creates a relatively small static site (vs. dynamic)

CollectionBuilder

Similar to Wax with different layouts

What is CollectionBuilder?

  • Grant funded project to create minimal computing exhibit off spreadsheet
  • Three versions, one is Github Pages focused others are more for repositories
  • Similar idea to Wax with a few added features for the simplest form
  • Lots of tutorial content which makes it beginner friendly
